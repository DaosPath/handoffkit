# Road to HandoffKit 1.0

0.8.0 focused on replayable workflow evidence and API stability. 0.9.0 is the
last pre-1.0 release focused on polish, compatibility checks, and documentation
completeness.

## 0.8.0

- Run traces for team, recipe, and tool execution reports.
- Replay summaries that never re-execute side effects.
- Report IO helpers.
- Stable API documentation.
- CLI diagnostics and report validation.

## 0.9.0

- Final public API audit and signature compatibility tests.
- Migration notes from 0.8.x to 0.9.x.
- Local wheel install smoke tests.
- Provider adapter malformed payload hardening.
- Documentation polish for the 1.0 surface.

## 1.0.0

- Stable public API commitment.
- Documented contracts.
- Reproducible workflow traces.
- Provider-neutral tool formats.
- PyPI package with clear examples and CLI diagnostics.
