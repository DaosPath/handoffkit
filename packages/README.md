# Packages

Workspace packages live here.

- `python/`: published Python package (`handoffkit` on PyPI).
- `contracts/`: shared JSON schemas and canonical fixtures used by Python and
  JavaScript tests.
- `core/`: JavaScript/TypeScript contract layer (`@handoffkit/core`) with
  async agents, tool format adapters, traces, and report helpers.
- `next/`: reserved for future Next.js helpers.
- `providers/`: reserved for future provider routing helpers.
