# HandoffKit Web

Next.js demo and documentation app for the HandoffKit monorepo.

Run commands from the repository root:

```powershell
pnpm web:dev
pnpm web:lint
pnpm web:build
```

Or from this folder:

```powershell
pnpm run lint
pnpm run build
pnpm exec next dev -p 3000
```

The app lives in `apps/web/src/app`.
